## 💡 Sobre

Conjunto de materiais para servidores públicos que já usam IA de forma superficial e querem
aproveitar melhor as ferramentas — com **segurança, método e senso crítico**. Os exemplos usam
**documentos públicos reais** de uma licitação de limpeza urbana para mostrar a IA aplicada ao
dia a dia da fiscalização.

> [!IMPORTANT]
> **Ideia central:** a IA é uma ferramenta de apoio — ela **não substitui** o servidor.
> Quem orienta a IA, revisa e assina é o profissional. **O dever de pensar não se delega.**

## ❓ O que os materiais respondem

1. Todas as IAs são iguais? Qual usar para cada tarefa?
2. Até onde posso subir documentos do trabalho na IA? É seguro?
3. Como ser específico nos comandos (prompts)?
4. Como tornar a IA um hábito diário no trabalho?

- Use a **apresentação** e os **vídeos** para estudar ou repassar à equipe.

## 🧭 Princípios que guiam os materiais

- 🎯 **Não existe "a melhor IA"** — existe a melhor para cada tarefa.
- 🔒 **Cuidado com dados** — anonimize o que for sensível; configure a privacidade.
- ✍️ **Prompt importa** — seja específico (Contexto · Papel · Formato · Especificidade).
- ✅ **Confira sempre** — a IA erra e inventa; a responsabilidade é do servidor.
